Capabilities

Everything for Mobile Maps

A complete mobile mapping engine in a single library — no cloud dependencies, no API keys, no vendor lock-in.

📦

Offline Tile Cache

LRU disk cache with configurable limits (256 MB default, 50K tiles). Pre-fetch entire regions for offline use. XYZ and MVT tile sources.

👆

Gesture Recognition

Multi-touch state machine handling pan, pinch-zoom, rotate, and tilt. Platform-agnostic — feed it touch events, get camera deltas.

🎨

Style Engine

Mapbox GL JSON-compatible styles with zoom-level interpolation. Fill, Line, Symbol, Circle, and Raster layer types.

🧭

Turn-by-Turn Navigation

On-device route tracking with step instructions, off-route detection (50m threshold), and arrival notification. No cloud required.

🔄

Offline Vector Editing

Local feature store with create/update/delete and sync status tracking. BBox spatial queries and GeoJSON export.

🗺️

Camera Model

Continuous zoom (0–22), bearing, pitch. Web Mercator projection with viewport-aware tile range calculation.

📍

Location Service

GPS abstraction with tracking modes (Follow, Heading, Course). Haversine distance and bearing calculations built in.

🖼️

Render Pipeline

Frame-based command buffer for platform GPU backends. Metal (iOS) and Vulkan (Android) ready. HiDPI/Retina aware.

tv_map_get_center_lon(state)Get center longitude

Platforms

Platform Integration

Native Swift and Kotlin examples for iOS and Android.

iOS — Swift Swift 
import TerraVista

class MapViewController: UIViewController {
    private var map: OpaquePointer?

    override func viewDidLoad() {
        super.viewDidLoad()
        let b = view.bounds
        let s = Float(UIScreen.main.scale)
        map = tv_map_create(UInt32(b.width),
                           UInt32(b.height), s)
        tv_map_set_center(map, 51.5074, -0.1278)
        tv_map_set_zoom(map, 14.0)
        tv_map_set_tile_url(map,
            "https://tiles.tiletopia.dev/{z}/{x}/{y}.mvt")
    }

    deinit { tv_map_destroy(map) }
}
Android — Kotlin Kotlin 
class MapView(ctx: Context) : View(ctx) {
    private var map: Long = 0

    init {
        System.loadLibrary("terravista_ffi")
        map = tvMapCreate(
            width.toUInt(),
            height.toUInt(),
            resources.displayMetrics.density)
        tvMapSetCenter(map, 51.5074, -0.1278)
        tvMapSetZoom(map, 14.0)
        tvMapSetTileUrl(map,
            "https://tiles.tiletopia.dev/{z}/{x}/{y}.mvt")
    }

    fun destroy() { tvMapDestroy(map) }

    // JNI bindings auto-generated from C header
    private external fun tvMapCreate(...): Long
    private external fun tvMapDestroy(s: Long)
}

Quick Start

Get Started in Minutes

Clone, build, and integrate into your mobile app.

Terminal bash 
# Clone
git clone git@github.com:GeoLang/terravista.git
cd terravista

# Build & test
cargo build
cargo test

# Build for iOS
rustup target add aarch64-apple-ios
cargo build --target aarch64-apple-ios -p terravista-ffi --release

# Build for Android
rustup target add aarch64-linux-android
cargo build --target aarch64-linux-android -p terravista-ffi --release
